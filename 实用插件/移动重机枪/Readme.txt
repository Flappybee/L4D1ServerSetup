指令
!minigun <type>    //在面前生成一个重机枪 type == 0 生成一代重机枪, type == 1 生成二代重机枪
!removeminigun     //删除你面前的重机枪


控制台变量
l4d2_moveablemachinegun                    //插件版本
l4d2_movablemachinegun_block               //搬运重机枪时禁止开枪  0 = 不禁止开枪 ,1 = 禁止开枪
l4d2_movablemachinegun_speed               //搬运重机枪时的行走速度    1.0 = 正常速度 , -0.1 = 不要修改
l4d2_movablemachinegun_distance            //多远距离内不可布置第二个  (0.0 禁用此功能, 默认200.0)

控制台变量：
l4d_bandw_notice                                 - 0 关闭通知，1 通知幸存者，2 通知所有人，3 通知感染。 默认值为 1。
l4d_bandw_type                                   - 0 打印聊天，1 显示提示框。 默认值为 1。
l4d_bandw_glow                                   - 0 打开发光，1 打开发光。 默认为 OFF。
